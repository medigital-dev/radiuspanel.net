Tutorial Instalasi Web Server/LAMP (Linux, Apache, MySQL, PHP)

Hal pertama yang harus diinstall dan di konfiguasi adalah apache2 dan mysql-server. Jika pada server anda sudah terinstall dan terkonfigurasi tanpa error, langkah ini bisa di skip. Jika belum silahkan mengikuti tutorial berikut.

# apt update
# apt install -y apache2 mysql-server php php-mysql
# mysql_secure_installation
y
0 #gunakan 0 supaya gak ribet passwordnya
radpass #password mysql sesuai keinginan anda
radpass #repeat password
y
y
y
y
y
Selanjutnya konfigurasi mysql-server
# mysql
> UNINSTALL PLUGIN validate_password;
> ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'radpass';
> FLUSH PRIVILEGES;
> exit;

Done. Instalasi dan konfigurasi mysql-server selesai
